Platform Overview

Mini AI Platform is a lightweight, fast AI chat interface built with HTML, Bulma CSS, and Vanilla JavaScript. It connects to a Knowledge Engine to provide intelligent responses based on your questions.

The platform features a clean, dark-themed interface similar to modern AI chat applications, with support for:

  • Real-time AI chat
  • Chat history persistence
  • Custom API key generation
  • Responsive design
  • Multiple LLM provider support

API Integration

The platform integrates with external knowledge services. Here's how to use the API:

Ask a Question

POST /api/answer

Request Body:

{"question": "What is artificial intelligence?"}

Response:

{ "answer": "Artificial Intelligence refers to..." }

Authentication

Include your API key in the request header:

Authorization: Bearer YOUR_API_KEY

Usage Instructions

1. Getting Started

Simply type your question in the chat input and press Enter or click the send button. The AI will process your question and provide a response.

2. Creating API Keys

Navigate to the "Create API" page to generate your own API key for external integrations.

3. Chat History

Your chat history is automatically saved in localStorage. You can view and continue previous conversations from the sidebar.

4. Settings

Click the settings button in the top navigation to customize:

  • AI name
  • Theme (dark/light)
  • Streaming responses
  • Sound notifications
  • LLM API keys
