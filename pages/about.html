Mini AI Platform

A lightweight AI web platform with knowledge engine integration. Simple, clean, and fast - inspired by the OpenAI chat interface.

AI Chat

Interactive chat interface with real-time responses. Ask questions and get intelligent answers instantly.

Knowledge Engine

Connects to knowledge servers to retrieve relevant information for your queries.

API Keys

Generate and manage API keys for external integrations and custom applications.

Simple Auth

One-click login with localStorage. No databases, no complex setup required.

Responsive Design

Works beautifully on desktop and mobile devices with a modern dark theme.

Modular Architecture

Clean, maintainable code with separate modules for chat, memory, and AI processing.

Connected Repositories

Knowledge Engine
The brain of the system that processes questions and retrieves relevant information. It acts as an intermediary between the platform and the knowledge server.
github.com/amkyawdev/amkyawdev-knowledge-engin
Knowledge Server
A storage system that holds knowledge files and provides data retrieval capabilities. The engine queries this server to get information for user questions.
github.com/amkyawdev/amkyawdev-knowledge-server
